## ✅ Tasks Completed

### Level 1 — Task 1: HTML/CSS
- Simple webpage with heading, paragraph, and image
- Basic form with input field and submit button
- Navigation menu with three links, hover effects, and smooth scroll
- Fully responsive layout with mobile media queries
- Aesthetic: Editorial magazine style with gold/cream palette

### Level 1 — Task 2: JavaScript
- **Color Shift Button** — cycles through a curated palette on each click
- **Time-Based Greeting** — detects morning/afternoon/evening/night and shows contextual message
- **Basic Calculator** — takes two inputs and adds them with live expression display
- All three components in a single dark-mode dashboard layout

### Level 3 — Task 1: Image Manipulation
- **CSS Image Gallery** — 9-image grid with hover zoom and overlay effects
- **Lightbox** — full-screen image viewer with prev/next navigation and keyboard support
- **Auto Slideshow** — 5 slides with auto-transition every 4 seconds, progress bar, dot nav
- Smooth transitions and accessible keyboard navigation

### Level 3 — Task 2: Landing Page
- Headline: "Web Developer Internship Opportunities at Cognifyz Technologies"
- Program overview and duration (8+ weeks)
- Highlights section: 6 key benefits in a hover-animated grid
- Qualifications: 8 desired skills with level indicators
- CTA buttons: "Apply Now" (links to email) and "Learn More"
- Footer with full contact info: email, website, LinkedIn, Instagram
- Hamburger mobile menu, scroll reveal animations, marquee ticker
